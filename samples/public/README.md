# Public Sample Workspace

This folder will hold lightweight, processed samples for 微信树洞 AI.

Rules:

- Do not store large public archives here.
- Do not store private WeChat chat logs here.
- Do not store copied dialogue lines unless the source license and privacy status are clear.
- Prefer synthetic seed examples and distilled behavior labels.
- Keep sticker and voice files out of this folder until asset rights are reviewed.

Large downloads that are currently deferred:

- U-Sticker sticker archives.
- EmotionTalk audio/video files.
- LCCC large archives.
- Any model weights.
