## Confirmed Product Direction

微信树洞 AI 是一个接入微信环境的 AI 陪聊智能体。第一版服务对象是你和少数朋友，主场景是生活压力、emo 和睡前陪伴。智能体人格是“嘴欠但靠谱的深夜损友”：能接梗、吐槽、陪你熬一会儿，也会在你明显上头时帮你刹车。

## Inferred Strategy

Use a hybrid strategy:

- Confirm product blueprint first.
- Build a web static prototype before formal coding.
- After the prototype is confirmed, define architecture and module boundaries.
- Then generate the master task board and begin implementation rounds.
